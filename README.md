# Skills

个人 AI Coding Skills 沉淀仓库，存放自定义技能，让 AI 助手在处理特定任务时遵循统一的规范和最佳实践。

## 已有 Skills

| Skill | 说明 |
|-------|------|
| [fastapi-backend-standard](./fastapi-backend-standard/SKILL.md) | 规范 Python FastAPI 后端项目结构、分层、测试和文档同步 |
| [git-commit-standard](./git-commit-standard/SKILL.md) | 规范 Git 提交信息格式、仓库初始化、.gitattributes 和 .gitignore 默认配置 |
| [python-path-standard](./python-path-standard/SKILL.md) | 规范 Python 项目文件路径写法，提供跨系统路径封装 |

每个 Skill 目录下至少包含一个 `SKILL.md`。

## 使用方式

将对应 Skill 的 `SKILL.md` 内容配置到 AI Coding 工具的 skills 目录中即可生效。
